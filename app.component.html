2.Property Binding

This is an example of Property Binding in which the property allowNewServer is initiated and the button is disabled for 2sec.
Property binding is used to bind the HTML to a particular property which will be updated dynamically

Property binding Vs String Interpolation

If we want to output something in the HTML template and print some text to it We use String interpolation.

String interpolation only works with normal template not with another expression of that template,mixing property binding and stroing interpolation will break the code of the app in angular.


if we want to change some property be that of HTML element or of any directive or a component we use property binding.
